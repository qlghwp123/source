식(expression)
    정의 : 엄밀한 것은 아니지만 다음 요소를 통틀어 말한다.
        변수
        상수
        변수나 상수를 연산자로 결합한 것
평가(evaluation)
    식의 값을 알아내는 것
    ex)
        int   52
        여기서 int 는 '형' 이라 하며 52는 '값' 이다.
        프로그램을 실행할 때 식이 평가된다. 식을 평가하면 형과 값을 얻을 수 있다.
매개변수
    함수에 전달되는 값을 저장하는 변수.
    형식매개변수(formal parameter)라 한다.
    함수를 호출할 때 사용하는 매개변수 값(value)을 '실인수(actual argument)'라 한다.
    간단하게 정의할 때는 '매개변수', 호출할 때는 '실인수'로 생각하면 된다.
알고리즘
    문제를 해결하기 위한 것으로, 명확하게 정의되고 순서가 있는 유한 개의 규칙으로 이루어진 집합.
